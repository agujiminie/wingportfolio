# Wing Zeng Portfolio

Single-page portfolio build for AI product design, motion-led interaction studies, and experimental digital craft.

## Stack

- React 19
- Vite 7
- Custom responsive CSS
- Canvas-based morph interaction

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Notes

- Optimized art assets are used for the primary scene renders to keep the production bundle lighter.
- Root-level export artifacts are ignored so the repo stays focused on the portfolio source.
